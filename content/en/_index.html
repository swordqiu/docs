The one cloud that contains all your clouds

Cloudpods manages not only on-premise KVM/baremetals, but also resources from many cloud accounts across many cloud providers. It hides the differences of underlying cloud providers and exposes one set of APIs that allow programatically interacting with these many clouds.

{{< /blocks/lead >}} {{< blocks/section type="features" color="white" >}} {{% blocks/feature icon="fas fa-server fa-3x" title="Private Cloud" height="auto" color="blue" %}} A light-weight private cloud that manages KVM hypervisor in scale. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-cube fa-3x" title="BareMetals" height="auto" color="blue" %}} A BareMetal cloud that automates the full life-cycle management of baremetal physical machines. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-cloudversify fa-3x" title="Multi-Cloud" height="auto" color="blue" %}} Multi-cloud management that is able to manage a wide range of cloud providers, including private cloud and public clouds. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-id-card fa-3x" title="Multi-Tenancy" height="auto" color="blue" %}} A complete multi-tenancy RBAC-enabled IAM (identity and access management) system. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-user-cog fa-3x" title="Cloud SSO" height="auto" color="blue" %}} Cloud SSO that allows accessing native webconsole of cloud providers with unified federated identities. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-code fa-3x" title="Unified APIs" height="auto" color="blue" %}} One set of feature-rich APIs to access a wide range of resources from heterogeneous cloud providers with consistent models and APIs. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/lead >}}
