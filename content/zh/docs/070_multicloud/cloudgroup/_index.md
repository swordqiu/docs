云用户组是公有云IAM用户的逻辑分组


## 平台支持

| 云平台(Provider)      | 是否支持|
| :---------------------| :-----|
| 阿里云(Aliyun)        | Y     |
| 腾讯云(Qcloud)        | Y     |
| 华为云(Huawei)        | Y     |
| 微软云(Azure)         | Y     |
| 谷歌云(Google)        | Y     |
| Aws(Aws)              | Y     |
| 优刻得(Ucloud)        | N     |
| 天翼云(Ctyun)         | N     |
| 移动云(Ecloud)        | N     |
| 京东云(JDcloud)       | N     |
